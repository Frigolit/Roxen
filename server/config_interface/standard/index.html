This is a pre-alpha view of the configuration interface

If your fonts are way to large (or small, depending on personal preferences), please adjust the unicode font size in your browser. This configuration interface uses UTF-8, and at least netscape has a diferent font size for unicode fonts (defaulting to 18) than normal fonts (defaulting to 12)

Edit -> Preferences -> Fonts -> (for the encoding ) Unicode (unicode-2-0) -> size
