Welcome to Roxen Challenger!

The first thing you will have to do is to set a user name and password. These will get you into the configuration interface when you want to configure the server later on.

If you have any questions or problems, first look at the incomplete FAQ, then check the mailing list archive (The roxen site is also searchable).

You should also read the online documentation in the configuration interface (this is an example of online documentation) If you still have a problem, just send a mail to the Roxen mailinglist and ask. You will be subscribed automatically if you send a mail to the list.

To get in touch with the developers, mail developers@roxen.com
