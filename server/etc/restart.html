Restarting server... 

Restart

 Roxen will restart automatically.

You might see the old process for a while in the process table when doing 'ps' or running 'top'. This is normal. Roxen waits for a while for all connections to finish, the process will go away after at most 15 minutes.

To return to the configuration interface, just press "back". If roxen is back online, you should get the previous view.
