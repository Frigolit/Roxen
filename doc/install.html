Installation Guide for AutoSite

  1. Install Roxen Challenger 1.3, for instance in /usr/www/roxen.

  2. Install the following source code trees, and add their paths to the module path in your newly installed Roxen Challenger:
    • AutoSite
    • logview_sql
    • intraseek
    • navigation
    • weblayout
    • menumaker

  3. Install Bind v.8 (available from http://www.isc.org/bind.html) and Mysql 3.22+ (available from http://www.tcx.se/)

  4. Setup LogView SQL:
    1. Edit the file logview_sql/LogViewSQL.pmod/config.h
    2. Change the definition of "DATABASE" to the correct value.
    3. Run logview_sql/tools/setup.pike

  5. Setup the SQL tables needed by AutoSite:
    Load the SQL table definitions found in AutoSite/doc/db.spec

  6. Install Infoflex SMS Gateway and Infoflex XFAX Gateway, and make sure their binaries are in the PATH of the web server.
